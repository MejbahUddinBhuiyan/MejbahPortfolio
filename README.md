# Mejbah Uddin Bhuiyan Portfolio

A modern full-stack academic and professional portfolio built with Laravel 12, PostgreSQL (Supabase), Cloudinary, Tailwind CSS, and Vite.

## Live Website

https://mejbahportfolio.onrender.com

## Features

- Responsive Modern UI
- Admin Dashboard
- Dynamic Profile Management
- Research Publication Showcase
- Project Portfolio
- Education & Skills Sections
- Blog Management System
- Certificate Showcase
- Gallery Management
- Contact Form
- Cloudinary Image Storage
- PostgreSQL Database (Supabase)
- Docker Deployment
- Render Hosting
